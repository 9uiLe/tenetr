修正提案 (focus.single-primary-decision)
『通知をオンにする』行をカード/chevron付きの独立操作から、テキストリンクや控えめな案内バナーへ格下げするか、初期表示から外して完了後の別画面/設定内に移動する。『次のパスは明日』文言と『ノーミス/ヒントなし』バッジも、完了の事実・連続記録の表示に含まれない付加情報であれば削除または折りたたみ、exemplar-v3同様に『アイコン→タイトル→ラウンド数→連続記録カード→単一CTA→軽いテキストリンク』へ縮約することを検討する。
