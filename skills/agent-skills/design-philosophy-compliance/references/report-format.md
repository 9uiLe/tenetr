# Agent 最終報告形式 (§11.3)

```text
実装内容
- 変更した画面
- 主な変更点

適用したデザイン原則
- 原則ID
- 実装への反映方法

評価結果
- Deterministic: Pass / Fail (fail 数と修正内容)
- Model-based: Pass / Warn (warn の内容)
- Human Review: 件数と各項目の要旨

成果物
- report.html
- evaluation.json
- run-manifest.json
```

Fail を修正せず報告しない。Warn / Human Review を省略しない。
